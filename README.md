# 📘 AksharaDeepa Tutor v2

AksharaDeepa Tutor v2 is an Android-based educational application designed to support self-learning for students, especially in rural and underserved areas. The app helps students track syllabus progress, practice quizzes, monitor weak areas, and improve learning consistency through a simple and user-friendly interface.

---

# 🚀 Features

## ✅ Syllabus Tracker
- Track chapter-wise completion
- Monitor subject progress
- Helps students stay organized

## 🧠 Quiz Mode
- Daily practice quizzes
- Multiple-choice questions
- Instant score evaluation

## 📊 Strength Analysis
- Visual progress tracking
- Identify weak and strong subjects
- Performance insights

## 🔔 Daily Goal Reminder
- Encourages consistent study habits
- Topic completion reminders

## 🌐 Offline Support
- Works without internet connection
- Local data storage using Room Database

## 🌍 Kannada Language Support
- Designed with rural accessibility in mind
- Regional language compatibility

---

# 🛠️ Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM + Clean Architecture
- **UI:** Jetpack Compose / XML
- **Database:** Room Database
- **Background Tasks:** WorkManager
- **Navigation:** Navigation Component
- **Design:** Material Design 3
- **Minimum SDK:** Android 7.0 (API 24)
